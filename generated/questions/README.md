# Generated Questions

This folder stores generated question banks for markdown chapters.

For MVP, questions are stored as JSON files before moving to a database.

## Question File Convention

Each chapter can have a matching question file:

```text
generated/questions/<course-slug>/<chapter-slug>.questions.json
````

Example:

```text
generated/questions/claude-101/s1c1-what-is-claude.questions.json
```

## Purpose

These question files will power:

* Practice quizzes
* Weak-area tracking
* Spaced repetition
* Review scheduling
* Future AI-generated question banks
